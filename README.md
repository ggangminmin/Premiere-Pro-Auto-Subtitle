# Premiere Pro 자동 자막 생성기

Premiere Pro 프로젝트(`.prproj`)의 **노래 플레이리스트 시퀀스**를 읽어서, 각 곡의
음성을 AI로 분석해 **시퀀스 타임라인에 정확히 맞춘 자막(SRT)** 을 자동으로 만들어 줍니다.
한국어(위) / 일본어(아래) 듀얼 자막 형식을 지원합니다.

> 동작 원리: `.prproj`(gzip XML) 파싱으로 곡 배치·타이밍 추출 → `faster-whisper`(GPU)로
> 줄별 타이밍 추출 → 가사 파일로 텍스트 교정 → SRT 생성.

---

## 무엇을 하나

1. `.prproj` 에서 지정한 시퀀스에 깔린 **오디오 클립(곡)** 들과 각자의 **타임라인 위치(시작/끝)** 를 추출
2. 각 곡 오디오를 **faster-whisper** 로 받아 **줄 단위 타이밍**을 뽑음 (NVIDIA GPU 가속)
3. 시퀀스 위치에 맞게 시간을 보정해 **일본어 SRT** 를 자동 생성
4. **번역 워크시트(JSON)** 를 같이 생성 → 한국어 칸을 채우면 **한일 통합 / 한국어 SRT** 까지 생성

---

## 준비물 (Prerequisites)

| 항목 | 설명 |
|---|---|
| **Python 3.10+** | https://www.python.org 에서 설치 |
| **(선택) NVIDIA GPU + 드라이버** | 있으면 훨씬 빠름. 없으면 자동으로 CPU 모드로 동작(느림) |
| **Premiere 프로젝트** | `.prproj` 파일. 자막 넣을 **시퀀스에 곡 오디오가 이미 배치**되어 있어야 함 |
| **오디오 파일** | 곡 wav/mp3 등 (프로젝트에서 경로를 찾지 못하면 `audio_dir` 로 지정) |
| **(선택) 가사 파일** | 곡 번호가 든 `.txt` (예: `교차점 1.txt`). 텍스트 교정/번역 참고용 |

### 네이밍 규칙
- 오디오 파일명과 가사 파일명 **끝의 숫자가 곡 번호**입니다. (예: `00_03AM 6.wav` ↔ `3am 6.txt` → 둘 다 6번)
- 이 숫자로 곡-가사를 매칭합니다.

```bash
git clone git@github.com:ggangminmin/Premiere-Pro-Auto-Subtitle.git
cd Premiere-Pro-Auto-Subtitle

```

## 사용법

### ⚡ 가장 간단한 방법 — 자동 설정 (다른 영상에도 그대로)
프로젝트 경로만 주면 **시퀀스·오디오·가사 폴더를 알아서 찾아** 실행합니다.
```bash
python run.py "C:/.../내프로젝트.prproj"
# 또는 프로젝트 폴더만 줘도 됨
python run.py "C:/.../내프로젝트폴더"
# 특정 시퀀스를 지정하려면
python run.py "C:/.../내프로젝트.prproj" "시퀀스 03"
```
- 오디오 클립이 가장 많은 시퀀스를 자동 선택하고, 설정을 `*_config.json` 으로 저장합니다.
- 다른 영상도 경로만 바꿔 같은 명령이면 끝. (아래 수동 설정은 세밀 조정용)

### 1) (수동) 설정 파일 만들기
`config.example.json` 을 복사해 `config.json` 으로 만들고 경로를 수정합니다.

```json
{
  "prproj": "C:/.../프로젝트.prproj",
  "sequence": "시퀀스 06",
  "audio_dir": "C:/.../sing list",
  "lyrics_dir": "C:/.../노래 가사",
  "output_dir": "C:/.../자막출력",
  "output_basename": "시퀀스06_자막",
  "language": "ja",
  "model": "medium",
  "device": "cuda"
}
```
- `sequence` 에 들어갈 정확한 이름을 모르면, 시퀀스 목록을 먼저 확인:
  ```bash
  python src/extract_layout.py "C:/.../프로젝트.prproj"
  ```

### 2) 실행
```bash
python run.py config.json
```
결과물(출력 폴더):
- `시퀀스06_자막_일본어.srt` — 일본어 자막 (자동)
- `시퀀스06_자막_translation.json` — **번역 워크시트**

### 3) 번역 (한국어 자막을 원할 때)
`_translation.json` 의 각 줄 `"ko"` 칸에 한국어 번역을 채우면 됩니다.

**가장 쉬운 방법 — 터미널의 AI 에게 시키기** (Claude Code 등):
> `시퀀스06_자막_translation.json 파일의 ja 를 한국어로 번역해서 ko 칸을 채워줘`

→ 별도 API 키 없이, 터미널에 있는 AI 가 번역해 줍니다.
(직접 번역하거나 ChatGPT 등에 붙여넣어도 됩니다.)

> 💡 한 줄(한 큐)에 가사 두 줄이 붙어 있으면, **`ja`·`ko` 양쪽 같은 위치에 ` / `** 를 넣어
> 주세요. SRT 생성 시 자동으로 별도 큐로 분리됩니다. (자세히: [docs/RULES.md](docs/RULES.md) §2)

그리고 **같은 명령을 다시 실행**하면 통합/한국어 SRT 가 추가로 생성됩니다.
```bash
python run.py config.json        # (또는 자동 설정 모드면 같은 프로젝트 경로로 재실행)
```
- `시퀀스06_자막_한일통합.srt` — 한국어(위)+일본어(아래)
- `시퀀스06_자막_한국어.srt` — 한국어만

## Premiere Pro 에 자막 넣기

**간단히 (1트랙):**
1. 시퀀스 열기 → `파일 > 가져오기` → `..._한일통합.srt` 선택
2. 프로젝트 패널의 자막을 캡션 트랙에 배치
3. 자막 클릭 → `기본 그래픽` / `캡션` 패널에서 폰트·크기·색·외곽선·위치 조정

**두 색으로 (한국어 흰색 위 / 일본어 노란색 아래):**
1. `..._한국어.srt` → 캡션 트랙1 (흰색, 외곽선 검정, 위쪽, 크게)
2. `..._일본어.srt` → 캡션 트랙2 (노란색, 외곽선 검정, 아래쪽, 약간 작게)
3. 스타일을 한 번 잡고 **[스타일 만들기]** 로 저장하면 전체에 일괄 적용

## 자막 형식 규칙
- 반복 추임새("Oh oh oh oh…")는 **2회로 축약**
- 한 자막에 여러 가사 줄을 합치지 않고 **줄마다 별도 자막**으로 분리
- 자세한 내용: [docs/RULES.md](docs/RULES.md)

## 한계 / 참고
- 노래(가창) 인식 특성상 줄 타이밍에 **1~2초 오차**가 있을 수 있습니다. 어색한 줄은 Premiere에서 살짝 밀어주세요.
- 가사에 한국어가 없으면 번역은 사람이 채우는 워크시트 방식입니다(자동 번역 미포함).
- 이 도구는 곡이 시퀀스 오디오 트랙에 배치된 음악/플레이리스트형 프로젝트를 가정합니다.
